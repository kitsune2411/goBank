# GoBank

GoBank is a Golang REST API for bank account management.

## Run Locally

Clone the project

```bash
  git clone https://github.com/kitsune2411/goBank.git
```

Go to the project directory

```bash
  cd goBank
```

Start the server

```bash
  make run
```
